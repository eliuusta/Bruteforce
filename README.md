# Bruteforce
Este projeto demonstra testes de força bruta realizados com Medusa em ambiente controlado utilizando Kali Linux e Metasploitable 2

Ataques de Força Bruta com Medusa e Kali Linux
Projeto Prático — DIO | Segurança da Informação

Este repositório contém minha documentação completa do desafio da DIO, no qual realizei ataques de força bruta utilizando o Kali Linux e a ferramenta Medusa, contra serviços vulneráveis expostos pela máquina Metasploitable 2.

O objetivo é demonstrar, na prática, como ataques de autenticação funcionam, como identificar credenciais fracas e como propor medidas de mitigação em um ambiente controlado.

 1. Objetivos do Projeto

Entender ataques de força bruta em FTP e SMB

Utilizar o Medusa no Kali Linux para auditoria

Realizar enumeração de usuários (SMB)

Documentar processos técnicos de forma clara

Propor medidas de mitigação

Compartilhar o projeto no GitHub como portfólio
